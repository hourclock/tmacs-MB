About tmacs[mb]

"tmacs[mb]" is a web-based tactile map automated creation system.
This system instantly creates an image of a tactile map, whitch in then printed on capsule paper and raised up by a heater.

Usage

Select the background maps from four types or none.

Bing
BING Map data.
mapbox
mapbox Map data.（Default）
GSI
Japan Geographical Survey Institute map data.
Satellite
Satellite Map data of GSI.

Transmittance of the background map can be changed in 100 steps from 0 to 100%. (default = 50) Adjust the transmittance so as not to disturb the tactile map.
Angle of rotation can be changed in 360 steps from -180 to 180 degree. (default = 0)

Select the edit modes from five types by editting.

Line
Draw a line.
Marker
Place circle or point marker.
Braille
Place a braille marker.
Direction
Place a direction marker.
Delete
Delete object by single click. Redo by double click.
